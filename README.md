# Progetto-Lab-IA
## Descrizione
Il progetto propone come obiettivo quello di realizzare un classificatore di generi musicali e in aggiunta un trasferimento di stile neurale tra due tracce audio. Per la classificazione sono stati utilizzati due approcci differenti, uno basato su Image Augumentation ed un altro che sfrutta le features delle tracce audio contenute in un file `.csv`.
